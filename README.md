***CodeAlpha_GenZ_AI_FAQ_Chatbot***

​An intelligent, desktop-based FAQ Chatbot application designed with a modern user interface and an integrated Natural Language Processing (NLP) matching core. This project utilizes text preprocessing, TF-IDF Vectorization, and Cosine Similarity to parse free-form user inputs and dynamically match them against an established knowledge base.
​Developed as part of the Software Engineering Internship at CodeAlpha.

---

***​🚀 Key Features***

​**🧠 Semantic NLP Matching Core:** Uses scikit-learn's TF-IDF Vectorizer and Cosine Similarity matrices to understand the underlying intent of a user's question, bypassing the limitations of rigid keyword matching.

​**🧹 Stop-Word Text Filtering:** Features automated preprocessing scripts that clean inputs and filter out common grammatical filler words to eliminate false-positive results.

​**📱 Responsive UI Grid Architecture:** Built with an advanced Tkinter structural design that scales components smoothly, maintaining visual balance when the window is maximized.

​**✨ Custom Unicode Render Module:** Bypasses default Windows/Tkinter text constraints to display high-resolution, sharp visual icons and emojis.

​**⚡ Interactive Suggestion Chips:** Implements direct-action shortcut buttons allowing users to seamlessly query core categories with a single tap.

---

​***🛠️ Tech Stack***

​**Language:** Python 3.x

​**Graphical User Interface:** Tkinter (Custom styled framework)

​**Machine Learning & NLP:** Scikit-Learn (TfidfVectorizer, cosine_similarity)

​**Text Processing:** Regular Expressions (re module)
